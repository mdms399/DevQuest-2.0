Terminal e Git

Conceitos Fundamentais

  • O que é o terminal
  • Linha de comando vs GUI
  • Por que usar o terminal

Aplicações Práticas

  • Navegação de pastas (cd, ls, pwd)
  • Executar programas
  • Gerenciar projetos
  • Instalar bibliotecas
  • Controle de versão com Git

Instalação e Configuração

  • Windows → Git Bash
  • Mac → Homebrew → brew install git
  • Linux → apt/dnf/pacman

Comandos Essenciais

  • pwd → diretório atual
  • ls → listar arquivos
  • cd → mudar diretório
  • mkdir → criar pastas
  • rm → remover arquivos/pastas
  • clear → limpar tela

Atalhos e Produtividade

  • Histórico: ↑ ↓
  • Autocompletar: Tab
  • Ctrl+C → parar comando
  • Ctrl+L → limpar tela
  • Ctrl+A → início da linha
  • Ctrl+E → fim da linha
